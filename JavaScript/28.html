    # 模块化

    # webpack

    1. 创建目录a,进入a ,npm init -y
    2. a/mod1.js
    export let a=12;
    1. a/index.js
    //引入模块
import * as mod1 from './mod1';
alert(mod1.a);
    1. a/index.html
    <!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<script type="text/javascript" src="build/bundle.js" ></script>
	</head>
	<body>
		
	</body>
</html>

    运行 报错，浏览器目前不支持es6模块化，这个时候需要webpack 5. a/创建webpack.config.js

    const path=require('path');

module.exports={
	entry: 'index.js',
	output: {
		path: path.resolve(__dirname, 'build'),
		filename: 'bundle.js'
	}
};
    1. 然后再a/执行 webpack
    2. 运行1.html,弹出12

    # 总结


    1. entry --入口地址
    2. output --输出 必须是个对象 path-绝对路径 filename--文件名
    3. mode--模式
    4. 所有的当前路径前面加./
